# BHR-Code-Global-Platform

Enterprise digital platform for BHR Code including corporate website, client portal, talent network platform, partner ecosystem, and knowledge hub.

## Repository Structure

- `frontend/` — Next.js web experience.
- `backend/` — Node.js microservices.
- `database/` — PostgreSQL schema and data assets.
- `infrastructure/` — Docker and Kubernetes configuration.
- `docs/` — architecture and planning documentation.
