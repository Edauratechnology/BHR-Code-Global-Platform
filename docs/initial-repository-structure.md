# Initial Repository Structure

This repository now includes the initial enterprise scaffolding:

- `frontend/` - Next.js application with core pages.
- `backend/` - Node.js microservices for API gateway and core domains.
- `database/` - PostgreSQL schema for foundational entities.
- `infrastructure/` - Docker and Kubernetes deployment configuration.
- `docs/` - Architecture and setup documentation.
